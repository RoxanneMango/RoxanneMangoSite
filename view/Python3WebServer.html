Python3 Webserver

A python3 webserver with HTML/CSS/JS frontend and MySQL database support. The webserver code is written for Linux but can run on windows as well. The main compatibility issues on windows are with the commands used in the Makefile (they are Linux specific). Fixing this problem is for a future update.
When starting the python script the server will attempt to connect to the MySQL database defined in the code (this should probably be moved to a config file of sorts ...). If the server cannot connect to the database it will terminate. After the backend has instantiated and successfully connected to the database, it will start the interactive console. The console has a few simple commands like 'help', 'reload', 'and shutdown/exit'. Then, as the last step the backend will start serving on the ip:port defined in the makefile. [ link to the Github repository ]


If the port is equal to 443 it will attempt to serve over HTTPS instead of HTTP, and use the SSL certificate and public key supplied in the config folder of the repository. You will need to supply this certificate and key yourself which requires you to have a valid domain name. It is possible to generate your SSL cert and key with Certbot, for example.
The frontend is minimal but it does allow the user to authenticate themselves via the login portal. When successfully logged in, the server will keep track of the session using cookies for a limited amount of time, after which the cookies expire and the user is logged out again.

Makefile:


.PHONY: test prod kill

IP = 127.0.0.1
PORT = 443
TEST_IP = 127.0.0.1
TEST_PORT = 4443

test:
	sudo python3 py3Backend.py $(TEST_IP) $(TEST_PORT)
prod:
	nohup sudo python3 py3Backend.py $(IP) $(PORT) &
kill:
	sudo pkill python3
