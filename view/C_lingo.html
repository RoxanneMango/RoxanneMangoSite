C_Lingo Webserver

C_Lingo is a C webserver with a minimal frontend that allows you to play the game Lingo in the browser. The frontend consists of a simple HTML/CSS/Javascript stack. The webserver code is written for Linux (no support for Windows or Mac environments). For the sockets the arpa/inet.h library was used; for the multithreading the pthread library.
The C_Lingo back-end code can be divided into two parts: The lingo_words command line application code, and the lingo server code. All the building of the code is automated via Makefiles.

LingoWords CLI application

Lingo_words is a C Command Line Interface (CLI) application used for gathering words with five, six, and seven letters. It saves the words in their respective .txt files. The lingo server code depends on lingo_words for its supply of words.

Test suite

As part of its test suite there is a test double and a C program that performs a static code analysis. The static code analysis counts the amounts of calloc() calls vs the amount of free() calls. If the two are not equal the test will fail.

Makefile:


CC = gcc
CFLAGS = -Wall -static -O3

OBJECTS = tests.o net_test.o file_test.o mem_test.o
EXEC = main

.PHONY: all clean run

all: $(OBJECTS)
	$(CC) -o $(EXEC) main.c $(OBJECTS) $(CFLAGS)

$(OBJECTS): %.o: %.c
	$(CC) -c $(CFLAGS) $< -o $@

clean:
	@rm *.o
	@rm $(EXEC)

run:
	@./$(EXEC)
